# 오버워치 캐릭터와 맞는 나의 연애유형 심리테스트 
jS와 css를 활용하여 나의 연애 유형과 비슷한 오버워치 캐릭터를 확인할 수 있습니다. 
평소 MBTI와 심리테스트에 관심이 많아, 사용자에게 재미를 더하고자 만들었습니다.
